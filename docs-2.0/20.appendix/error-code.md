# 错误码

Nebula Graph 运行出现问题时，会返回错误码。本文介绍错误码的详细信息。

!!! note

    - 如果出现错误但没有返回错误码，或错误码描述不清，请在[论坛](https://discuss.nebula-graph.com.cn/)或 [GitHub](https://github.com/vesoft-inc/nebula/issues) 反馈。
    - 返回`0`表示执行成功。
