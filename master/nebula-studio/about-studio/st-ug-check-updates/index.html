    版本更新¶

    Studio 处于持续开发状态中。用户可以通过 Studio 发布版本更新 查看最新发布的功能。

    成功连接 Studio 后，用户可以在页面右上角点击版本号，再点击 新发布，前往查看 Studio 的版本更新记录。
